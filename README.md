## Overview

This repository contains the prediction results from two major computational tools, **SWISS-MODEL** (Homology Modeling) and **I-TASSER** (Threading/Ab Initio Hybrid), used to model the 3D structure of the **SARS-CoV-2 3C-like proteinase (Mpro/NSP5)**. Mpro is a vital antiviral drug target, and this analysis compares high-quality models generated by different state-of-the-art platforms.

The target sequence for both models is the full Mpro protein (306 residues), labeled `>7LYH_1|Chain A...mutated.fasta` in the SWISS-MODEL report.

## Setup and Visualization

To visualize the predicted structures and analyze their differences:

1.  **Clone the repository:**
    ```bash
    git clone [Your Repository URL]
    cd [Your Repository Name]
    ```

2.  **View 3D Structure:**
    Load the **`7LYHmodel_01.pdb`** file into any molecular visualization software (e.g., **PyMOL** or **ChimeraX**).

    * Compare the SWISS-MODEL's predicted conformation against the top models reported in the **`I-TASSER results.pdf`** to assess the structural consensus.
